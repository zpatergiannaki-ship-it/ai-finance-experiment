Μελέτη Χρηματοοικονομικού Βοηθού ΤΝ

Σας προσκαλούμε να συμμετάσχετε σε μια ακαδημαϊκή έρευνα που διεξάγεται από τη Ζωή Πατεργιαννάκη, στο πλαίσιο μεταδιδακτορικής έρευνας στο Πανεπιστήμιο Πειραιώς.

Η έρευνα εξετάζει τον τρόπο με τον οποίο οι άνθρωποι αλληλεπιδρούν με συστήματα τεχνητής νοημοσύνης σε χρηματοοικονομικά σενάρια λήψης αποφάσεων.

Κατά τη διάρκεια της έρευνας θα σας ζητηθεί:

  • να διαβάσετε σύντομα χρηματοοικονομικά σενάρια
  • να αλληλεπιδράσετε με έναν βοηθό τεχνητής νοημοσύνης
  • να απαντήσετε σε ορισμένες σύντομες ερωτήσεις σχετικά με την εμπειρία σας

Για τη συμμετοχή σας στη μελέτη πρέπει να είστε άνω των 18 ετών.

Η συμμετοχή σας είναι απολύτως εθελοντική. Δεν απαιτείται καταγραφή προσωπικών δεδομένων που να μπορούν να σας ταυτοποιήσουν και όλες οι απαντήσεις συλλέγονται ανώνυμα.

Τα δεδομένα θα χρησιμοποιηθούν αποκλειστικά για ερευνητικούς και ακαδημαϊκούς σκοπούς. Σε περίπτωση δημοσίευσης ή παρουσίασης των αποτελεσμάτων σε επιστημονικά περιοδικά ή συνέδρια, δεν θα συμπεριληφθούν πληροφορίες που θα μπορούσαν να αποκαλύψουν την ταυτότητά σας.

Τα δεδομένα θα αποθηκευτούν σε ασφαλή ερευνητική υποδομή και θα χρησιμοποιηθούν μόνο για σκοπούς ακαδημαϊκής ανάλυσης.

Η συνολική διάρκεια συμμετοχής στη μελέτη είναι περίπου 10–15 λεπτά.

Δεν υπάρχουν γνωστοί κίνδυνοι που να συνδέονται με τη συμμετοχή στη μελέτη και μπορείτε να αποχωρήσετε οποιαδήποτε στιγμή χωρίς καμία συνέπεια.

Δεν προβλέπεται οικονομική αμοιβή για τη συμμετοχή στη μελέτη.

Εάν έχετε οποιεσδήποτε ερωτήσεις σχετικά με τη συμμετοχή σας ή τη διεξαγωγή της έρευνας, μπορείτε να επικοινωνήσετε με την ερευνήτρια:
