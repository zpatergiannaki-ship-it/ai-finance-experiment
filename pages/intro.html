Μελέτη Χρηματοοικονομικού Βοηθού Τεχνητής Νοημοσύνης

Σας προσκαλούμε να συμμετάσχετε σε μια ακαδημαϊκή έρευνα που διεξάγεται από τη Ζωή Πατεργιαννάκη, στο πλαίσιο μεταδιδακτορικής έρευνας στο Τμήμα Οικονομικής Επιστήμης του Πανεπιστημίου Πειραιώς.

Η έρευνα εξετάζει πώς οι άνθρωποι αλληλεπιδρούν με συστήματα τεχνητής νοημοσύνης σε χρηματοοικονομικά σενάρια λήψης αποφάσεων.

Κατά τη διάρκεια της έρευνας θα σας ζητηθεί:

  • να διαβάσετε σύντομα χρηματοοικονομικά σενάρια
  • να αλληλεπιδράσετε με έναν βοηθό τεχνητής νοημοσύνης
  • να απαντήσετε σε σύντομες ερωτήσεις σχετικά με την εμπειρία σας

Για τη συμμετοχή σας στη μελέτη πρέπει να είστε άνω των 18 ετών.

Η συμμετοχή σας είναι απολύτως εθελοντική. Δεν θα συλλεχθούν προσωπικά δεδομένα που μπορούν να σας ταυτοποιήσουν.

Τα δεδομένα θα χρησιμοποιηθούν αποκλειστικά για ερευνητικούς και ακαδημαϊκούς σκοπούς. Σε περίπτωση δημοσίευσης ή παρουσίασης των αποτελεσμάτων σε επιστημονικά περιοδικά ή συνέδρια, δεν θα συμπεριληφθούν πληροφορίες που θα μπορούσαν να αποκαλύψουν την ταυτότητά σας.

Τα δεδομένα θα αποθηκευτούν σε ασφαλή ερευνητική υποδομή και θα χρησιμοποιηθούν αποκλειστικά για σκοπούς επιστημονικής ανάλυσης.

Η συνολική διάρκεια συμμετοχής στη μελέτη είναι περίπου 15 λεπτά.

Δεν υπάρχουν γνωστοί κίνδυνοι που σχετίζονται με τη συμμετοχή στη μελέτη. Μπορείτε να αποχωρήσετε οποιαδήποτε στιγμή χωρίς καμία συνέπεια.

Δεν προβλέπεται οικονομική αμοιβή για τη συμμετοχή στη μελέτη.

Εάν έχετε οποιεσδήποτε ερωτήσεις σχετικά με τη συμμετοχή σας ή τη διεξαγωγή της έρευνας, μπορείτε να επικοινωνήσετε με την ερευνήτρια:
